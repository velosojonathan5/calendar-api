Requisitos da aplicação Calendar

Deve ser possível cadastrar usuários, sendo que deve ser validado se o 
formato do email e do telefone estão corretos. Também não deve ser validado 
se o email já foi cadastrado

Deve ser possível alterar os dados do usuários.

Deve ser possível buscar os usuários, filtrando por nome ou email

Deve ser possível deletar um usuários, deletar também suas tarefas

Deve ser possível cadastrar uma tarefas para um usuário

Deve ser possível buscar as tarefas de um usuário

Deve ser possível deletar as tarefas de um usuário

## Descrição

Aplicação base para o workshop de Node.JS no Bootcamp 2021 da Squadra Digital.
Utiliza Nest.JS como framework web.

## Instalação

```bash
$ npm install
```

## Rodando o app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
